# Task Spark (NodeJS + PostgreSQL) Backend

Task Spark is simply a Todo Full-Stack web application project which improves the productivity of the user by allowing a user to enter list of tasks, mark them completed, edit task, delete task, etc.

In this backend project i have used NodeJS as JavaScript runtime environment, JavaScript as programming language, PostgreSQL as Database using pg npm package, ExpressJS as NodeJS web framework, jsonwebtoken, etc.

This project is only the backend part of Task Spark. Below is the frontend part of this project build with Vite, ReactJS, Redux, JavaScript, etc technologies.
